# Write-up Support \[Easy\] - Hack The Box (Informal)
\\
**Mi perfil de Hack The Box: [julianglez16](https://app.hackthebox.com/users/1084561)**

Con smbclient listamos todas las shares que hay de samba. Existe una que se llama "support-tools" que contiene varios programas y zips, entre ellos putty.


Con un disassembler puedes sacar una contraseña que se usa para extraer información del ldap. 


Una vez extraida se ver que el usuario "support" tiene un campo "info: Ironside47pleasure40Watchful". Con evil-winrm se prueba a hacer la conexión con usuario support y dicha contraseña y obtenemos conexión con la máquina con powershell.
